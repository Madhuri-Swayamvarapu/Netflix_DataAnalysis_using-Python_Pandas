# 🎥 Netflix_DataAnalysis_using-Python_Pandas

This project focuses on analyzing the Netflix Movies and TV Shows dataset using Python and Pandas. The dataset contains detailed information about Netflix content such as movies, TV shows, directors, cast, country, release year, ratings, duration, and genres.

The main objective of this project is to perform data cleaning, preprocessing, and exploratory data analysis (EDA) to extract meaningful insights from the dataset.

**Key Features**
★ Data Cleaning and Preprocessing

★ Handling Missing Values

★ Date and Time Conversion

★ Filtering and Conditional Analysis

★ Country-wise Content Analysis

★ Movie vs TV Show Analysis

★ Duration and Rating Analysis

★ Data Visualization and Insights

**Technologies Used**

• Python

• Pandas

• Matplotlib

• Seaborn

• Jupyter Notebook


**Learning Outcomes**

Through this project, the following concepts were practiced:

Data manipulation using Pandas

String operations and filtering

Datetime conversion

Conditional indexing

Splitting and transforming columns

Frequency analysis using value_counts()

Exploratory Data Analysis techniques

**Conclusion**

This project helps in understanding how real-world datasets are cleaned, transformed, and analyzed to generate meaningful business insights using Python.
